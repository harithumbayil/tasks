1. GIT
    Git is a distributed version control system, meaning it tracks changes in files over time and allows multiple developers to collaborate on a project simultaneously without overwriting each other's work. It's essentially a way to manage the history of your project and collaborate effectively
  2. HTML
    HTML, which stands for HyperText Markup Language, is the foundation of the World Wide Web, defining the structure and content of web pages
    It's a markup language, not a programming language, that uses tags and elements to organize text, images, and other multimedia content for display in a web browser.
  3. CSS
    Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML)
  4. Basic tags
    Basic HTML tags are fundamental building blocks for structuring web pages. They define the content and layout of a webpage, instructing web browsers how to render text, images, links, and other media. These tags are enclosed in angle brackets and usually come in pairs: an opening tag and a closing tag.
